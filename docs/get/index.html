Getting Started

parsed-path tagged location to parse your pathname This example creates two simple pathname, a Root and a File, with some parsed attachment to it:

const Root = parsed`/`;
const File = Root`home``user``dir``
  name: file;
  ext: .txt;
`;
render(
  <>
    <Root/>
    <File/>1
  </>
);
//home/user/file.txt1

Utilities#

const Mount = parsed`foo``bar``baz`.mount`foo``bar`; // to be 'foo/bar/foo/bar/baz'
const From = parsed`foo``bar``baz`.from`foo``bar`; // to be 'baz'
const To = parsed`foo``bar``baz`.to`foo``bar`; // to be '..'
render(
  <>
    Mount: <Mount/><br/>
    From: <From/><br/>
    To: <To/><br/>
  </>
)
Mount: foo/bar/foo/bar/baz
From: baz
To: ..
