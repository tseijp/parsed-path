# Getting Started

parsed-path tagged location to parse your pathname
This example creates two simple pathname,
a Root and a File, with some parsed attachment to it:

import parsed from 'parsed-path'
import {Live} from '../components/Live'

<Live.Card scope={{ React, parsed }} code={`
const Root = parsed\`/\`;
const File = Root\`home\`\`user\`\`dir\`\`
  name: file;
  ext: .txt;
\`;
render(
  <>
    Root: <Root/> <br/>
    File: <File/>
  </>
);
`.trim()}/>

### Utilities

<Live.Card scope={{ React, parsed }} code={`
const Mount = parsed\`foo\`\`bar\`\`baz\`.mount\`foo\`\`bar\`; // to be 'foo/bar/foo/bar/baz'
const From = parsed\`foo\`\`bar\`\`baz\`.from\`foo\`\`bar\`; // to be 'baz'
const To = parsed\`foo\`\`bar\`\`baz\`.to\`foo\`\`bar\`; // to be '..'
render(
  <>
    Mount: <Mount/> <br/>
    From: <From/> <br/>
    To: <To/>
  </>
);
`.trim()}/>

### React Recipies

<Live.Card scope={{ React, parsed }} code={`
const VERSION = 2.1;
const API = parsed\`api\`\`v\${VERSION}\`\`user\`\`\${props => props.user}\`;
const As = props => React.createElement('button', props, props.href);
render(
  <API as={As} user={10}/>
)
`.trim()}/>
